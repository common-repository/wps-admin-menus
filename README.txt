Select the visability of administration menus for different user roles in your WordPress website. Restrict user roles from the entire backend / admin area.

== Description ==

Allow an Administrator to hide / show backend menus from the different user role types. Restrict specified user roles that cannot access the admin / backend area at all. 
All menus added by additional plugins can be controlled by this plugin.
Menus that are default hidden from certain role types will not be changed. 
<br/>
Requires <a href='https://wordpress.org/plugins/wps-complete/'>WPS Complete</a> to be installed. It is the central hub to our plugin suite.
<br/>

Our full suite of plugins can be viewed at <a href='https://wpscomplete.com/plugins/'>WPS Complete Plugins</a>


== Installation ==

1. This plugin requires the WPS Complete plugin to be installed <a href="https://wordpress.org/plugins/wps-complete/" target="_blank">WPS Complete</a>
1. Upload WPS admin menus to wp-content/plugins
1. Click "Activate" in the WordPress plugins menu
1. Select the WPS Complete navigation, select the Admin Menus submenu option
